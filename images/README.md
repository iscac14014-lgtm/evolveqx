Place the hero image file here so the site can use it as the hero background.

- Save the image you selected (the attached image named by you) as: `images/hero.jpg`
- Recommended formats: `jpg` or `webp`.
- Recommended size: at least 1800px wide, optimized for web (compress to ~200-400KB).

Example (Windows):
1. Open the attachment image in your browser.
2. Right-click > Save image as...
3. Save to `c:\xampp\htdocs\dashboard\evolveQX\images\hero.jpg`

After placing the file, reload the page to see the new hero background.
